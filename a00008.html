Detailed Description

Parameters
context_id- circuit instance identifier. returned by createCircuitContext
document_dir- directory where files will be read from or written to .
json_string- a JSON object used to de-serialize (re-construct) the respective object. Probably generated by their serialize function.
Returns
0 : success
-1 : invalid context_id
1 : error occurred , get the error description with getLastFunctionMsg
