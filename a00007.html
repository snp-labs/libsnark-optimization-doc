Detailed Description

The following functions work only with the embedded circuit.
Circuits constructed with arith/inputs files are not supported.
They update a key/value (Primary input name/BigInteger) map maintained by the circuit constructor.

Parameters
context_id- circuit instance identifier. returned by createCircuitContext
input_name- primary input name (or description) as key
value- new value
value_str- new value : a BigIngeter value in hexadecimal string
array_index- index of a primary input array to update
input_json_string- a JSON object, with keys as primary input names and values as BigInteger in a hexadecimal format.
Returns
0 : success
-1 : invalid context_id
1 : invalid input_name
1 : array_index out-of-range
