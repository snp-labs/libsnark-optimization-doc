int createCircuitContext ( const char *  circuit_name,
int  proof_system 
)

Initialize a circuit context.

The API allows users to create multiple circuits concurrently.
Use this function to create a separate circuit instance.

Parameters
circuit_name- the name of the circuit. Also selects which of the embedded circuit constructor to use.
proof_system- select the proof system : R1CS_GG or R1CS_ROM_SE.
Returns
An identifier to the circuit instance.
